# SOS Terra

Projeto React + Vite pronto para subir na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Subir na Vercel

- Crie um repositório com estes arquivos
- Importe o projeto na Vercel
- A Vercel deve detectar automaticamente como **Vite**
- Build command: `npm run build`
- Output directory: `dist`

## Observação

O conteúdo do ecossistema local pode ser ajustado diretamente em `src/App.jsx`.
